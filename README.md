## 0) Start here (Setup suggestions)

It helps to do these steps first. Many other offers require you to have the GitHub pack already active.

1. **Get verified on GitHub** - Apply here: https://github.com/education/students
2. **Claim the GitHub Student Developer Pack** - Once verified, go here: https://education.github.com/pack
3. **Organize your email** - It is helpful to make a specific folder in your inbox for "Student Perks" so you don't lose renewal emails or license keys.

## 1) GitHub Student Developer Pack

This is usually considered the most important pack to get. One verification here unlocks benefits on many other sites.

**Claim here:** https://education.github.com/pack

**Key Highlights:**
- **GitHub Copilot:** AI coding assistant (Free).
- **Microsoft Azure:** $100 credit for cloud services.
- **JetBrains:** Free access to professional IDEs like IntelliJ and PyCharm.
- **Namecheap:** 1 year free domain name ending in `.me`.
- **DigitalOcean:** $200 credit for new users (valid 1 year).
- **Heroku:** Credits for hosting apps (typically $13/mo for 24 months).
